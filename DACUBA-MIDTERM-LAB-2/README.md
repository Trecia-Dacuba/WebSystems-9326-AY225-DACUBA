# GFG Linux Academic Scraper
### Automated Learning Material Generator — GeeksforGeeks Linux Topic

---

## 📁 Project Structure

```
gfg_scraper/
├── app.py              ← Flask web server (run this!)
├── scraper.py          ← Web scraper logic (BeautifulSoup)
├── pdf_generator.py    ← PDF builder (ReportLab)
├── requirements.txt    ← Python dependencies
├── templates/
│   └── index.html      ← Dashboard UI
├── data/
│   └── articles.json   ← Auto-created: cached scraped data
└── output/
    └── *.pdf           ← Auto-created: generated PDFs
```

---

## 🚀 Setup Instructions (Do This Once)

### Step 1 — Make sure Python is installed
Open a terminal / command prompt and run:
```bash
python --version
```
You need **Python 3.10 or higher**. Download from https://www.python.org if needed.

---

### Step 2 — Go into the project folder
```bash
cd gfg_scraper
```

---

### Step 3 — Create a virtual environment (recommended)
A virtual environment keeps project packages separate from your system Python.

**Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

**macOS / Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
```

You'll see `(venv)` at the start of your terminal prompt — that means it's active.

---

### Step 4 — Install all dependencies
```bash
pip install -r requirements.txt
```

This installs:
| Package | Purpose |
|---|---|
| `flask` | Web server / dashboard |
| `requests` | HTTP requests for scraping |
| `beautifulsoup4` | HTML parsing |
| `lxml` | Fast HTML parser (used by BS4) |
| `reportlab` | PDF generation |

---

### Step 5 — Run the app
```bash
python app.py
```

You should see:
```
============================================================
  GFG Linux Academic Scraper — Dashboard
  Open http://127.0.0.1:5000 in your browser
============================================================
 * Running on http://0.0.0.0:5000
```

---

### Step 6 — Open the dashboard
Open your browser and go to:
```
http://127.0.0.1:5000
```

---

## 🖥️ How to Use the Dashboard

### 1. Start Scraping
- Click **"▶ Start Scraping"**
- A progress bar shows each article being fetched
- Scraping takes a few minutes (polite 2s delay between requests)
- Data is saved automatically to `data/articles.json`

### 2. Preview Articles
- Click **"Load Articles"** to see scraped content as cards
- Click any card to open a detailed modal with all extracted fields

### 3. Generate PDF
- Enter your name in the text field
- Click **"⬇ Generate PDF"**
- A download link appears instantly
- PDF is saved in the `output/` folder

---

## ⚠️ Troubleshooting

| Problem | Fix |
|---|---|
| `ModuleNotFoundError` | Run `pip install -r requirements.txt` again |
| `Address already in use` | Change port: `app.run(port=5001)` in `app.py` |
| Scraper returns 0 articles | GFG may have changed their HTML — check your internet connection first |
| PDF looks blank | Make sure you ran the scraper first and have cached articles |
| `venv\Scripts\activate` not found (Windows) | Use PowerShell, or try `venv\Scripts\activate.bat` |

## 📋 robots.txt Compliance

The scraper follows GFG's `robots.txt` by:
- **2-second delay** between every request
- **Descriptive User-Agent** identifying the scraper
- **Skipping** `/login`, `/profile`, and other disallowed paths
- **Not** re-scraping pages repeatedly

---

## 🔄 Re-running the Scraper

The `data/articles.json` file is a local cache. You can:
- Click **"↻ Refresh Info"** to see cache status without re-scraping
- Click **"🗑 Clear Cache"** to delete the cache and re-scrape fresh
- Generate the PDF any time from the existing cache — no internet needed

---

## 📄 PDF Output Format

The generated PDF includes:
- **Cover Page** — Title, date, subject category, article count
- **Table of Contents** — Numbered list of all articles
- **Per-Article Sections** (one per article):
  - Colour-coded difficulty badge
  - Key Technical Concepts
  - Code Snippets (dark monospace block)
  - Complexity Analysis
  - References / Related Links
- **Header** — Subject category + generation date on every page
- **Footer** — Page number + "Generated by [Your Name]" on every page
